# 🔍 نمای کلی پروژه

---

## 🎯 هدف پروژه چیست؟

هدف اصلی توسعه یک سیستم تحلیلی هوشمند برای:

- درک رفتار مشتریان
- پیش‌بینی فروش آینده
- کشف ناهنجاری‌ها
- کمک به تصمیم‌گیری مدیریتی

این پروژه نمونه‌ای از پیاده‌سازی عملی مفاهیم علم داده در یک محیط واقعی است.

1️⃣ دریافت داده خام

4️⃣ پیش‌بینی  
5️⃣ خوشه‌بندی  
6️⃣ تشخیص ناهنجاری  
7️⃣ نمایش در داشبورد وب
